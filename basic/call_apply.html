call 和 apply 的模拟实现

1. call()方法在使用一个指定的this值和若干个指定的参数值的前提下调用某个函数或方法。


var foo = {
  value: 1,
}

function bar() {
  console.log(this.value);
}

bar.call(foo); //1

2.1 call改变了this的指向，指向foo。

2.2 bar函数执行了。

3.模拟实现第一步

试想当调用call的时候，把foo对象改造成如下：


var foo = {
  value: 1,
  bar: function() {
    consoe.log(this.value);
  }
}

foo.bar(1);

虽然这样是实现了指向问题，但是却给foo添加了属性，这可不行 不过也不用担心，我们用delete在删除它就好了

4.模拟的步骤可以分为：

  • 将函数设为对象的属性
  • 执行该函数
  • 删除该函数

根据这个思路，实现第一版 call2 函数


//第一版
Function.prototype.call2 = function(context) {
 //首先要获取调用call2函数，用this可以获取
  context.fn = this;
  context.fn();
  delete context.fn;
}

  • 2.函数是可以有返回值的

7.apply的模拟实现


Function.prototype.apply2 = function(context, arr) {
  var context = Object(context) || window;
  context.fn = this;

  var result;
  if(!arr) {
   result = context.fn();
  } else {
   var args = [];
   for (var i = 0, len = arr.length; i < len; i++) {
    args.push('arr[' + i + ']');
   }
    result = eval('context.fn(' + args + ')');
  }

  delete context.fn;
  return result;
}

apply() 的实现基本跟call() 的实现是差不多的

  apply(fn, [arg1, arg2])
  call(fn, arg1, arg2)

区别是参数的传值方式不一样
